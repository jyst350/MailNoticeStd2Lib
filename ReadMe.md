# MailNoticeStd2Lib

基于.NET Standard 2.0 的邮件通知库。

支持异步特性。

## HTML邮件模板变量

```

$DISPLAYHEADERINFO 邮件模板正文的页头信息

$DISPLAYSUBJECT 邮件模板正文标题

$CONTEXT 邮件模板正文内容

$DISPLAYFOOTERINFO 邮件模板正文的页脚信息

$COLOR 邮件模板正文的页头信息背景颜色
```
